# Responsive-PhotoGallery

A simple, clean, and fully responsive photo gallery built with HTML and CSS. Perfect for showcasing your photography portfolio or any image collection without heavy frameworks.

<img width="1862" height="897" alt="image" src="https://github.com/user-attachments/assets/81c449fa-d23d-45b8-9ce6-eef87e48e774" />

## Features

- Fully responsive grid layout (works on mobile, tablet, and desktop)  
- Pure HTML + CSS (no JavaScript required)  
- Lightweight and fast loading  
- Modern CSS Grid + Flexbox design  
- Hover effects with smooth scaling and overlay captions  
- Easy to customize colors, spacing, and layout  
- Supports any number of images
